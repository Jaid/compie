## CLI Usage
After installing package `compie` globally, you can use its command line interface.
```bash
compie
```
For usage instructions:
```bash
compie --help
```




## Development



Setting up:
```bash
git clone git@github.com:jaid/compie.git
cd compie
npm install
```
Testing:
```bash
npm run test:dev
```
Testing in production environment:
```bash
npm run test
```
